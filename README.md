<p class=MsoNormal><b>Custom Products API Plugin<o:p></o:p></b></p>

<p class=MsoNormal>A WordPress plugin for integrating with a remote WooCommerce
API to manage products. This plugin allows WordPress administrators to list,
create, and configure products on a remote WooCommerce site directly from the
WordPress admin dashboard.<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal><b>Features<o:p></o:p></b></p>

<ul style='margin-top:0in' type=disc>
 <li class=MsoNormal style='mso-list:l4 level1 lfo1;tab-stops:list .5in'><b>List
     Products</b>: Fetch and display products from a remote WooCommerce store
     in a table format (ID, Name, Price, Stock Status).<o:p></o:p></li>
 <li class=MsoNormal style='mso-list:l4 level1 lfo1;tab-stops:list .5in'><b>Create
     Products</b>: Add new products to the remote WooCommerce store using an
     intuitive form.<o:p></o:p></li>
 <li class=MsoNormal style='mso-list:l4 level1 lfo1;tab-stops:list .5in'><b>API
     Configuration</b>: Manage API credentials and endpoint URLs via a
     user-friendly settings page.<o:p></o:p></li>
 <li class=MsoNormal style='mso-list:l4 level1 lfo1;tab-stops:list .5in'><b>Secure
     Integration</b>: Implements authentication with WooCommerce API using
     Consumer Key and Consumer Secret.<o:p></o:p></li>
 <li class=MsoNormal style='mso-list:l4 level1 lfo1;tab-stops:list .5in'><b>Error
     Handling</b>: Displays success or error messages for API operations.<o:p></o:p></li>
</ul>
